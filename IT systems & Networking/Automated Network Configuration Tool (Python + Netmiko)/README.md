 
# Network Automation Toolkit

Python-based network automation toolkit for MikroTik routers using Netmiko.

## Features

- Automated router configuration backup
- VLAN deployment automation
- Network compliance auditing

## Technologies

- Python
- Netmiko
- SSH automation
- MikroTik RouterOS

## Usage

Backup router configuration

python netauto.py backup

Deploy VLANs

python netauto.py vlan

Run compliance audit

python netauto.py audit
